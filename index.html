Neural-net-designer

Why not use a neural network to decide the topology of a neural net?

This experimental project is to train a neural network to design another neural net.

High level design

We use a predictor network to decide, for a given node in a network (input and output connectivity, weights, learning rate parameters), the utility of that node to the networks loss function.

Then the network is trained for a while both with and without a particular node. If the node adds to the networks accuracy, the predictor network is trained with a positive output, otherwise that result is trained with a negative output.

New nodes shall be added to the network periodically based on sampling highly positive outputs from the predictor network.

The network shall be regularly checkpointed, and reverted to checkpoints whenever loss regresses.

Implementation

Not yet started. The nature of this design makes it suboptimal for parallelization on GPU's etc, so it will be implemented for toy models only.
